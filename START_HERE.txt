DEPLOY TO CLOUD (10 MINUTES)
════════════════════════════

1. Create GitHub repo: https://github.com/new
   Name: nse-scanner

2. Upload files:
   git clone https://github.com/YOUR_USERNAME/nse-scanner.git
   cd nse-scanner
   # Copy server_v2.py, index.html, requirements.txt, Procfile, .gitignore
   git add .
   git commit -m "NSE scanner v2"
   git push origin main

3. Deploy on Render (FREE):
   • Go to https://render.com
   • Click "New +" → "Web Service"
   • Connect GitHub repo
   • Name: nse-scanner
   • Build: pip install -r requirements.txt
   • Start: python server_v2.py
   • Click "Create Web Service"

DONE! Your scanner is live at:
https://nse-scanner.onrender.com

═══════════════

Q: Port 5000 already in use?
A: pkill -f "python server"

Q: Only 88 stocks showing?
A: Make sure you're using server_v2.py, not old server.py

Q: TradingView data blocked?
A: Works fine on Render. On local, yfinance fallback works.
